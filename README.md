# TelegramBot

A simple [Telegram Bot API](https://core.telegram.org/bots/api#available-methods) wrapper for elixir.

## Installation

Just add `telegram_bot` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:telegram_bot, "~> 0.1.0"}
  ]
end
```

## Methods supported

1. `getMe`
2. `sendMessage`
